Registration

Features

CompileS sideNav contains different Language options , File options,
search bar for syntax.

Editor

Friendly user editor with syntax highlighting when desired programming language is selected in nav bar. The selected language is extension for the editor content. Provide input in the editor space provided exclusively for input.

File

Various file options are being provided.
1 Sharing file.
2 Upload file from PC.
3 Upload file from Github.
4 Upload file from Gist.
5 Download
6 Delete

Upload File

You can upload file from your pc. Select extension and continue execution.

Upload File From Github

You can upload file from Github Repo no need to clone the Repo Jsut enter File URL located in Github.

Upload File From Gist

You can upload file by giving Gist URL (RAW) and also any URL.

Download File

You can download the editor content. Content will be downloaded without actual extension provided at CompileS. Content will be in myFile.txt.
For better results open .txt from any IDE.

Delete

Confirm before deleting the content. Can’t get back the content once deleted. No trash is being maintained.

Search

Suggestions are given with respect to the language selected. Function name,description and example are provided for reference. Maximum built-in functions are covered.

Clear

click clear button to clear the output screen else the content as a whole will be provided as input for next execution.
